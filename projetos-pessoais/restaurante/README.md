# 🍔 Sistema de Restaurante - Algoritmo em Visualg

## 📌 Descrição do Projeto
Este projeto consiste em um algoritmo desenvolvido na linguagem de pseudocódigo do **Visualg**, com o objetivo de simular o funcionamento de um sistema simples de pedidos de um restaurante.

O sistema permite que o usuário escolha itens de um cardápio, informe a quantidade desejada, visualize um resumo do pedido e finalize a conta escolhendo diferentes formas de pagamento.

---

## 🎯 Objetivo
O objetivo do projeto é praticar conceitos fundamentais de lógica de programação, como:

- Estruturas condicionais (`se`, `escolha`)
- Estruturas de repetição (`repita`, `enquanto`)
- Variáveis
- Entrada e saída de dados
- Operações matemáticas
- Controle de fluxo

---

## 🍕 Funcionalidades do Sistema

O sistema possui as seguintes funcionalidades:

### 📋 Cardápio
O usuário pode escolher entre os seguintes itens:

- Hambúrguer
- Pizza
- Refrigerante
- Suco
- Sobremesa

Cada item possui um valor definido no início do algoritmo.

---

### 🛒 Sistema de Pedido
- O cliente escolhe o número do produto.
- Informa a quantidade desejada.
- O sistema adiciona o item ao pedido.
- O valor é acumulado na conta total.

O cliente pode continuar fazendo pedidos até decidir fechar a conta.

---

### 🧾 Resumo do Pedido
Ao finalizar os pedidos, o sistema exibe:

- Produtos escolhidos
- Quantidade de cada item
- Subtotal por produto
- Valor total da conta

---

### 💰 Taxa de Serviço
O cliente pode optar por adicionar **10% de taxa de serviço** ao valor total da conta.

---

### 💳 Formas de Pagamento

O sistema oferece quatro formas de pagamento:

1. **Débito**
   - Pagamento direto sem taxa.

2. **Crédito**
   - Acréscimo de 1% na conta.
   - Possibilidade de parcelamento em até 3 vezes.

3. **Pix**
   - Desconto de 5% no valor final.

4. **Dinheiro**
   - Permite inserir o valor pago.
   - Calcula o troco automaticamente.
   - Caso o valor seja insuficiente, permite complementar o pagamento.
