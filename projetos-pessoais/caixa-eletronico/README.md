# 🏧 Caixa Eletrônico em Visualg

Projeto de simulação de um **Caixa Eletrônico** desenvolvido em **Portugol** utilizando o Visualg.
O sistema permite que um usuário realize operações bancárias básicas através de um menu interativo no terminal.

---

## 📌 Funcionalidades

O sistema possui as seguintes funcionalidades:

* 🔐 **Login com validação de usuário e senha**
* 🔁 **Controle de tentativas de login**
* 💰 **Consulta de saldo**
* 💵 **Depósito de dinheiro**
* 💸 **Saque de dinheiro**
* 🔄 **Transferência para outro destinatário**
* 📄 **Visualização de extrato**
* 🔑 **Alteração de senha**
* 🚪 **Encerramento do sistema**

---

## 👤 Credenciais padrão

Para acessar o sistema:

```
Login: lulu
Senha: 123
```

O sistema permite até **3 tentativas de login** antes de bloquear o acesso.

---

## 🧠 Conceitos de programação utilizados

Este projeto foi desenvolvido com o objetivo de praticar **lógica de programação**, utilizando conceitos como:

* Variáveis
* Estruturas condicionais (`se`, `senao`)
* Estruturas de repetição (`repita ... ate`)
* Estrutura de escolha (`escolha ... caso`)
* Operações matemáticas
* Controle de fluxo
* Entrada e saída de dados (`leia`, `escreval`)

---

## 📊 Operações do sistema

Após o login, o usuário terá acesso ao seguinte menu:

```
1 - Ver saldo
2 - Depositar dinheiro
3 - Sacar dinheiro
4 - Transferir
5 - Ver extrato
6 - Alterar senha
7 - Sair
```

Cada operação possui validações para evitar erros como:

* valores negativos
* saldo insuficiente
* confirmação de operações

---

## 📄 Extrato

O sistema registra:

* Quantidade de depósitos
* Total depositado
* Quantidade de saques
* Total sacado
* Quantidade de transferências
* Total transferido
