# 📘 Lógica de Programação

Este repositório contém meus estudos de **lógica de programação**, organizados de forma progressiva, com exemplos práticos, desafios e pequenos projetos.

O objetivo é construir uma base sólida para desenvolvimento de software, cobrindo desde conceitos básicos até estruturas mais avançadas.
